Política de Privacidade

Última atualização: 05 de Dezembro de 2025

1. Uso de Dados do Strava

O aplicativo LeRunners utiliza a API do Strava para importar dados de atividades físicas (como distância, tempo e ritmo) exclusivamente para fins de acompanhamento técnico pelo treinador da equipe.

2. Armazenamento

Os dados importados são armazenados de forma segura em nosso banco de dados e são visíveis apenas para o atleta e para a comissão técnica da equipe LeRunners.

3. Não-Compartilhamento

Nós não vendemos, alugamos ou compartilhamos seus dados do Strava com terceiros, anunciantes ou outras empresas.

4. Seus Direitos

Você pode revogar o acesso do LeRunners à sua conta Strava a qualquer momento através das configurações do Strava ou solicitando a exclusão de seus dados ao administrador do clube.

Este aplicativo é "Powered by Strava" e segue as Diretrizes de Marca da API do Strava.
